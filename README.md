## Installation

### 1. Installer Termux dépendances

```pkg update -y && pkg upgrade -y pkg install python git mpv portaudio termux-api```

#

### 2. Cloner le projet

```git clone https://github.com/milfortmarcandy/QUEEN-AI.git cd QUEEN-AI```

#

### 3. Installer Requirements et les outis automatiquement

```pip install -r requirements.txt && bash install.sh```

#

## Requirements

```requests edge-tts SpeechRecognition PyAudio```
